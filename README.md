# Smart Expense Tracker for Students
Smart Expense Tracker is a web-based application that helps students manage daily expenses, set monthly budgets, and analyze spending patterns using charts and reports.

User Registration and Login

Add / Edit / Delete Expenses

Monthly Budget Setting

80% Budget Alert

Pie Chart & Bar Chart Reports

Frontend:

HTML
CSS
Bootstrap
JavaScript
Chart.js

Backend:

Node.js
Express.js

Database:

MySQL

Installation Guide:

git clone https://github.com/GLakshika/smart-expense-tracker/pull/new/develop
cd smart-expense-tracker

cd server
npm install

node server.js or npm start

server/config/db.js
